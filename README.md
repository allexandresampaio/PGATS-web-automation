# Projeto de Automação Web com Cypress

Este projeto tem como objetivo automatizar testes de aplicações web utilizando o framework Cypress.

## Objetivo
Automatizar cenários de testes end-to-end, garantindo a qualidade e o funcionamento das funcionalidades principais da aplicação alvo.

## Estrutura
- Os testes estão localizados na pasta `cypress/e2e`.
- As fixtures e dados de apoio ficam em `cypress/fixtures`.
- Configurações e comandos customizados podem ser encontrados em `cypress/support`.

## Como executar
1. Instale as dependências:
	```bash
	npm install
	```
2. Execute os testes:
	```bash
	npx cypress run
	```
	ou, para abrir a interface interativa:
	```bash
	npx cypress open
	```

## Requisitos
- Node.js instalado
- Cypress instalado como dependência do projeto
